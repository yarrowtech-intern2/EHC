# EHC Platform

Phase 1 foundation for the EHC healthcare SaaS platform.

## Apps

- `apps/web`: Next.js web application
- `apps/api`: NestJS API
- `apps/mobile`: Expo mobile application
- `supabase`: database schema and migration assets

## Phase 1 scope

- Auth foundation
- Tenant and facility model
- Admin tenant management dashboard
- Shared design tokens and responsive shells
- Audit log and platform data scaffolding
